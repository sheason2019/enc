# 连接至服务器

在用户档案页面，点击服务器配置按钮即可配置当前账号的服务器信息。

![档案页面](./profile_page.png)

点击后应用会跳转到如下页面。

![服务器配置页面](./service_page.png)

点击右上角的新建按钮，会弹出输入服务器 URL 的表单。

这里我以 ENC 的公共测试服务器 *http://enc-svr.sheason.site* 为例。
